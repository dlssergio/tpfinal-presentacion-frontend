FrontEnd

Desarrollo de Sistemas Web

Objetivo

E-commerce SPA

Desarrollo de una aplicación web moderna que consume una API REST real para mostrar productos tecnológicos.

  • Navegación dinámica con React Router.
  • Filtrado por categorías (Celulares, Computación).
  • Diseño Responsivo con Bootstrap.

Stack Tecnológico

Selección de herramientas basada en la documentación de la cátedra.

React & Hooks

Uso de useState y useEffect para ciclo de vida y estados locales.

Axios & API

Consumo asíncrono de datos desde DummyJSON con manejo de errores (Try/Catch).

Context API

Gestión de Estado Global para el Carrito de Compras.

Formik & Yup

Manejo y validación robusta de formularios.

Experiencia de Usuario

Carrito Persistente

Los productos seleccionados se guardan en localStorage, permitiendo al usuario recargar sin perder su compra.

Feedback Visual

Implementación de SweetAlert2 para notificaciones de éxito y validación de formularios en tiempo real.

Proyecto Finalizado

Aplicación desplegada y lista para su uso.
